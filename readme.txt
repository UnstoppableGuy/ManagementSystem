УСТАНОВКА И ЗАПУСК ПРИЛОЖЕНИЯ

1. Запустите setup.exe
2. Следуйте инструкциям установщика
3. После установки запустите приложение через ярлык

ЕСЛИ ПРИЛОЖЕНИЕ НЕ ЗАПУСКАЕТСЯ ИЛИ ПОЯВИЛАСЬ ОШИБКА :
"Не удалось подключиться к LocalDB"

РЕШЕНИЕ:
1. Скачайте SQL Server Express:
2. Выберите "Express" и установите с компонентом "LocalDB"
3. Перезапустите компьютер
4. Запустите приложение снова

ЛОГИ:
Папка Logs/ рядом с программой — если возникнут ошибки
